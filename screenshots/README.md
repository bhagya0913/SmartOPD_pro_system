## 📊 Role-Based Access

The system provides role-based dashboards with specific functionalities:

### Patient
- View and book appointments
- Access medical records and test results
- View prescriptions
- Submit feedback

### Doctor
- View patient queue and appointment list
- Access patient medical history
- Record diagnosis and treatment
- Issue prescriptions and referral forms
- Request lab tests

### Pharmacist
- View prescriptions
- Dispense medicines
- View prescription history

### Diagnostic Technician
- View pending test requests
- Update test status
- Upload test results (PDF/images)

### Receptionist
- Verify patient arrival
- Register new patients
- View daily queue
- Override quota/time limits

### Admin
- Manage staff
- Configure OPD hours and quotas
- Set closed dates
- Generate reports
- View feedback and audit logs
